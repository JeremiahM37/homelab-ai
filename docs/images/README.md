# Screenshots & assets

This directory holds the images referenced from the project README.

Suggested set (capture once and replace the placeholders below):

| File | What to capture |
|------|-----------------|
| `dashboard.png` | The Home tab of the PWA with 4-6 real services rendered |
| `chat-tool-calls.png` | The Chat tab mid-stream — tool-call card visible, then answer underneath |
| `config-editor.png` | The Config tab showing a couple of sections expanded |
| `history.png` | The History tab with the Scans sub-tab showing a few entries |
| `architecture.svg` | Optional — the architecture diagram from docs/architecture.md re-rendered as SVG |

Recommended capture settings:
- iPhone 14 Pro viewport (393 × 852) in Chrome devtools for the mobile shots
- Desktop 1280×800 for the dashboard hero
- PNG, 2x DPR for crispness on README cards

Run the project locally with `homelab-ai demo` for a populated dataset
without needing a real homelab.
